# Simply Switch – Versicherungsmakler

> **Dein unabhängiger Versicherungsmakler** – Privatkunden, Beamte & Selbstständige  
> 🌐 Live: [semkodi.github.io/versicherung](https://semkodi.github.io/versicherung)

---

## ✨ Features

- 🏠 **Startseite** – Hero-Bereich, Zielgruppen, Warum Simply Switch, Bewertungen, FAQ
- 👥 **Privatkunden** – Haftpflicht, Vorsorge, Hausrat, Kfz-Versicherung
- 🎖️ **Beamte** – Dienstunfähigkeit, PKV, Beihilfe, Referendariat
- 💼 **Selbstständige** – Betriebshaftpflicht, Inhalts-, Flotten- & Rechtsschutzversicherung
- 📊 **Bedarfs-Radar** – interaktiver Versicherungs-Bedarf-Check
- 📬 **Kontaktformular** – direkte E-Mail-Weiterleitung
- ⚠️ **Schaden melden** – eigene Unterseite für Schadensmeldungen
- 📱 **Vollständig responsiv** – optimiert für Desktop, Tablet & Mobile

## 🚀 Lokale Entwicklung

```bash
# Repository klonen
git clone https://github.com/Semkodi/versicherung.git
cd versicherung

# Abhängigkeiten installieren
npm install

# Entwicklungsserver starten
npm run dev

# Produktions-Build erstellen
npm run build
```
